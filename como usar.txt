Por ahora solo se puede probar el server del horoscopo

Abrir una terminal
- Generar las clases de java:
$ javac ServidorCentral.java
$ javac ServidorHoroscopo.java
$ javac Cliente.java

- Ejecutar el servidor:
$ java Servidor

- Abrir otra terminal en el mismo directorio
- Ejecutar el servidor horoscopo:
$ java ServidorHoroscopo

- Abrir otra terminal en el mismo directorio
- Ejecutar el cliente:
$ java Cliente
- Ingresar texto y presionar enter
Permite ingresar cualquiera de los signos del horoscopo, si se ingresa algo distinto muestra mensaje de error
Escribir exit para finalizar o ctrl+clases
